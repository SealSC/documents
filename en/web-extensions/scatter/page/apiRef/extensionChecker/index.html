Methods¶

installed¶

async installed()

Description

Check if Scatter web extension was installed.

Returns :

Promise<Result<Bool, Status>>:

Boolean value true will be set into the data field of Result if Scatter installed.

isMainnet¶

async isMainnet()

Description

Check if Scatter connected to mainnet.

Returns :

Promise<Result<Bool, Status>>:

If Status is SUCCESS , the data field of Result will be true if Scatter connected to EOS mainnet.

isLogin¶

async isLogin()

Description

Check if Scatter is logged in.

Returns :

Promise<Result<Bool, Status>>:

If Status is SUCCESS , the data field of Result will be true if Scatter was login.
