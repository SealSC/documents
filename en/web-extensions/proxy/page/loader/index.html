module method¶

load¶

async load(extension, extra)

Description

Load extension to proxy.

Parameters :

extension - ExtensionWrapper:

ExtensionWrapper instance to load.
extra - Any:
Config data required by extension.

Returns :

Promise<Result<ExtensionProxy, Status>>:

ExtensionProxy instance will be set into the data field of Result if success.
