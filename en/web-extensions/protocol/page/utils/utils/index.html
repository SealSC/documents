mulWithPow¶

mulWithPow(original, base, pow = 1)

Description

original * Math.pow(base, pow)

Parameters :

original - String | Number:

The multiplier

base - String | Number:

The base number

pow - String | Number:

The power number

Returns :

String:

Result string

Example¶

mulWithPow(2, 10, 3) // 2 * Math.pow(10, 3)
> "2000"
